# Deep-Learning---Based-Weed-and-Crop-Detection-in-Agricultural-Fields
This project develops an automated deep learning-based system to detect and localize weeds and crops in agricultural field images. It also estimates weed density and provides recommendations to support efficient and sustainable weed management.
